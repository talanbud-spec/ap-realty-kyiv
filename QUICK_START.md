# ⚡ Швидкий старт — 5 хвилин до живого сайту

## 🎯 Ваша мета
Розмістити сайт на Netlify і отримати постійне посилання для гранту.

## 📦 Крок 1: Завантажте архів (1 хв)
- Завантажте `ap-realty-kyiv.zip`
- Розпакуйте його (Extract All)
- Ви отримаєте папку `ap-realty-kyiv`

## 🌐 Крок 2: Реєстрація на Netlify (2 хв)
1. Перейдіть на https://www.netlify.com
2. Натисніть **"Sign up"** (верхній правий кут)
3. Виберіть **"Sign up with GitHub"** або **"Sign up with email"**
4. Дотримуйтесь інструкцій

## 📤 Крок 3: Розміщення (1 хв)
1. Натисніть **"Add new site"**
2. Виберіть **"Deploy manually"**
3. **Перетягніть папку `ap-realty-kyiv`** в область "Drag and drop"
4. Чекайте 30 секунд — готово!

## 🎉 Крок 4: Отримайте посилання (1 хв)
Netlify дасть вам посилання типу:
```
https://ap-realty-kyiv.netlify.app
```

**Це ваше постійне посилання для гранту!**

---

## 🆘 Якщо щось не працює

| Проблема | Рішення |
|----------|---------|
| Сайт не завантажується | Перевірте, що папка містить `index.html` |
| Netlify не приймає папку | Перетягніть саму папку, не ZIP архів |
| Помилка при розміщенні | Перевірте консоль браузера (F12) |
| Посилання не працює | Чекайте 1-2 хвилини, потім оновіть сторінку |
