# 📋 Інструкція розміщення сайту на Netlify

Цей документ описує, як розмістити сайт **AP Realty Kyiv** на Netlify та підключити власний домен.

## 🚀 Крок 1: Реєстрація на Netlify

1. Перейдіть на https://www.netlify.com
2. Натисніть **"Sign up"**
3. Виберіть **"Sign up with GitHub"** (або email)
4. Дотримуйтесь інструкцій реєстрації

## 📦 Крок 2: Розміщення сайту

### Варіант A: Drag & Drop (найпростіше)

1. Завантажте всі файли сайту в папку:
   - `index.html`
   - `app.js`
   - `styles.css`
   - `netlify.toml`
   - `README.md`

2. На Netlify перейдіть до **"Deploys"**
3. Перетягніть папку в область **"Drag and drop"**
4. Netlify автоматично розмістить сайт
5. Ви отримаєте посилання типу: `https://ap-realty-kyiv.netlify.app`

### Варіант B: GitHub Integration (для постійних оновлень)

1. Завантажте файли на GitHub
2. На Netlify натисніть **"New site from Git"**
3. Виберіть **GitHub** як провайдер
4. Виберіть репозиторій `ap-realty-kyiv`
5. Netlify автоматично розміщує сайт при кожному оновленні

## 🌐 Крок 3: Підключення власного домена

1. На Netlify перейдіть до **"Domain settings"**
2. Натисніть **"Add custom domain"**
3. Введіть ваш домен: `ap-realty-kyiv.com.ua`
4. Netlify покаже інструкції для вашого реєстратора доменів
5. Оновіть DNS записи у вашого реєстратора (HostPro, Namecheap тощо)
6. Чекайте 24-48 годин для поширення DNS

## 🔒 Крок 4: Увімкнення HTTPS

1. На Netlify перейдіть до **"Domain settings"**
2. Знайдіть **"HTTPS"**
3. Натисніть **"Provision certificate"**
4. Netlify автоматично видасть SSL сертифікат (безплатно)

## 📝 Крок 5: Налаштування редиректів (опціонально)

Файл `netlify.toml` вже налаштований для:
- ✅ Редиректу всіх запитів на `index.html` (для SPA)
- ✅ Кешування статичних файлів
- ✅ Безпеки заголовків

## 🔄 Оновлення сайту

### Якщо ви використовуєте GitHub:
1. Оновіть файли у GitHub репозиторії
2. Netlify автоматично розміщує нову версію

### Якщо ви використовуєте Drag & Drop:
1. Завантажте оновлені файли
2. Перетягніть папку на Netlify
3. Сайт оновиться

## 🆘 Розв'язання проблем

### Сайт не завантажується
- Перевірте, що всі файли розміщені
- Очистіть кеш браузера (Ctrl+Shift+Delete)
- Перевірте консоль браузера на помилки (F12)

### Домен не працює
- Перевірте DNS записи
- Чекайте 24-48 годин для поширення
- Перевірте налаштування у реєстратора доменів

### Статичні файли не завантажуються
- Перевірте шляхи до файлів у `index.html`
- Переконайтесь, що файли знаходяться в тій же папці
- Перевірте консоль браузера на помилки 404
